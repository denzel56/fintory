# React Forms

- Keep form values typed.
- Validate user input at form boundaries.
- Show field-level errors where possible.
- Disable or guard duplicate submissions.
- Reset form state deliberately after success or modal close.
- With Mantine, prefer `useForm` unless the project has chosen another form library.
- Do not mix validation and persistence logic in UI components.
