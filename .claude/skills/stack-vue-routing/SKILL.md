# Vue Routing

- Keep routes discoverable and consistently named.
- Use layout routes for shared page chrome.
- Parse and validate route params at boundaries.
- Use navigation guards only for real routing concerns.
- Avoid using routes as hidden global state.
- Expected Fintory screens include projects, dashboard, imports, transactions, categories, and analytics.
