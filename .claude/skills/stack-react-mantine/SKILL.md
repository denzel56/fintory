# React Mantine UI

- Prefer Mantine primitives before custom UI.
- Use `AppShell`, `Stack`, `Group`, `Grid`, `SimpleGrid`, and `Container` for layout.
- Use Mantine form controls consistently.
- Use `Modal`, `Drawer`, `Menu`, `Popover`, and `Notification` patterns instead of ad hoc overlays.
- Use `Table` or a chosen data-table solution consistently for transactions.
- Keep spacing, radius, colors, and typography aligned with the theme.
- Ensure desktop and small-window layouts remain usable.
- Avoid custom CSS when Mantine props or theme tokens solve the problem cleanly.
