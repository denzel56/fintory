# Vue Forms

- Keep form values typed.
- Validate at input and submit boundaries as appropriate.
- Show field-level errors where possible.
- Guard duplicate submissions.
- Reset form state deliberately after success or modal close.
- Use VeeValidate or the project's chosen form library for complex forms; keep simple forms simple.
- Do not mix validation and persistence logic in UI components.
