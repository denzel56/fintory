# Vue Stack

- Use Vue 3 Composition API and `<script setup lang="ts">` by default.
- Type props and emits explicitly.
- Keep templates readable; move complex expressions to computed values or functions.
- Extract reusable logic into composables.
- Prefer local `ref`/`reactive` state before global stores.
- Keep component responsibilities narrow and clear.
