# Web Documentation Research

- Prefer official documentation first.
- Include the library version in queries when known.
- Use web search only when repository context and built-in knowledge are insufficient.
- Summarize the relevant API, not the whole page.
- Do not use web examples blindly; adapt them to the repository patterns.
