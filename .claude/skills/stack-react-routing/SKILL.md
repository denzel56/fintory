# React Routing

- Keep route definitions discoverable and consistent.
- Use layout routes for shared page chrome.
- Keep route params typed or parsed at boundaries.
- Avoid using routing as hidden global state.
- Preserve direct navigation and reload behavior where the platform supports it.
- Expected Fintory screens include projects, dashboard, imports, transactions, categories, and analytics.
