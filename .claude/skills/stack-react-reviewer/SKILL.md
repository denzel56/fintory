# React Reviewer

- Hooks follow React rules and dependency expectations.
- JSX is readable and not overloaded with business logic.
- Component props are typed and meaningful.
- State is placed at the smallest useful scope.
- Mantine usage is consistent with the theme and existing UI.
- Interactive controls have labels, disabled/loading states, and keyboard-friendly behavior where relevant.
- UI does not hide CSV import errors or database failures.
